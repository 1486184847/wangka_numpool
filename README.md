### 如何使用

1.安装最新版本Python 3

2.安装依赖包：flask, requests

3.运行main.py文件，在浏览器中打开[http://127.0.0.1:5000/](http://127.0.0.1:5000/)，选择好省市，点击操作->启动爬虫即可

### Tips

爬虫每3秒获取一次号码列表，联通每次会随机返回100号码，与前面获取的结果会有重复，因此增长速度会越来越慢，当100条内新增条目低于10条时，爬虫自动终止。
